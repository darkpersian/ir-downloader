# 📦 IR-Downloader

یک ابزار قدرتمند مبتنی بر GitHub Actions برای دانلود فایل‌ها از لینک مستقیم، دسته‌بندی خودکار، رمزگذاری و ذخیره‌سازی دائمی در مخزن گیت‌هاب.  
با این روش میتونید با نت ملی لینک یا فایل هاتون رو دانلود کنید.

🔗 **لینک حمایت:** [https://daramet.com/sleeeeeeeeeep](https://daramet.com/sleeeeeeeeeep)

---

## ✨ قابلیت‌های کلیدی

- 🔗 دانلود از هر لینک مستقیم (HTTP/HTTPS)
- 🧠 تشخیص خودکار نوع فایل → ویدئو / صدا / تصویر / سند / سایر
- 📁 ساختار هوشمند پوشه‌ها
- 🔐 رمزگذاری دلخواه (اختیاری) با 7-Zip
- ✂️ تقسیم خودکار به قطعات **۹۰ مگابایتی** (قابل تغییر نیست)
- 💾 ذخیره دائمی در خود مخزن (نه Artifact موقتی)
- 📥 دانلود یکجا با کلیک روی Code → Download ZIP
- 🔄 عدم تداخل فایل‌های قبلی – هر فایل در پوشه جداگانه
- ⏱️ بدون محدودیت زمانی (برخلاف Artifactهای ۹۰ روزه)

 ## 💡 برای شروع – دریافت لینک مستقیم

فایل خود را در تلگرام به یکی از ربات‌های تبدیل فایل به لینک بفرستید و لینک دانلود رو دریاقت کنید.

---

## ⚙️ مراحل گام به گام (فقط چند دقیقه)

### ۱. فورک (کپی) کردن پروژه

- به لینک زیر بروید:  
  🔗 [https://github.com/ali25az/ir-downloader](https://github.com/ali25az/ir-downloader)
- روی دکمه **Fork** (بالای راست صفحه) کلیک کنید.
- در صفحه جدید، روی **Create Fork** بزنید.
- چند ثانیه صبر کنید تا یک کپی از پروژه در اکانت شما ساخته شود.

### ۲. وارد اکانت فورک شده خود شوید

بعد از فورک، به‌طور خودکار به صفحه پروژه کپی شده هدایت می‌شوید.

### ۳. اجرای عملیات دانلود

- روی تب **Actions** کلیک کنید.
- در سمت چپ، روی **「📦 دانلود و ذخیره در مخزن」** کلیک کنید.
- دکمه آبی **Run workflow** را بزنید.
- کادری باز می‌شود. در آن:
  - **لینک مستقیم** فایل مورد نظر را وارد کنید (در صورت چند لینک، با کاما `,` جدا کنید).
  - (اختیاری) رمز فایل فشرده را وارد کنید.
- سپس روی دکمه سبز **Run workflow** کلیک کنید – فرآیند دانلود شروع می‌شود.

### ۴. چگونه بفهمیم دانلود تمام شده است؟

- بسته به حجم فایل، ممکن است چند دقیقه تا چند ساعت طول بکشد.
- برای پیگیری پیشرفت:
  - دوباره به تب **Actions** بروید.
  - روی همان اجرا (run) کلیک کنید.
  - روی جعبه **download-and-commit** کلیک کنید تا جزئیات و لاگ را ببینید.
- وقتی کار آن تمام شد، کنار نام اجرا یک **دایره سبز** ظاهر می‌شود.  
  (نیازی نیست صفحه را باز نگه دارید، می‌توانید بعداً برگردید.)

### ۵. دریافت فایل دانلود شده

- بالای صفحه، روی نام مخزن اصلی (مثلاً `ir-downloader`) کلیک کنید.
- روی دکمه سبز **Code** کلیک کنید.
- گزینه **Download ZIP** را بزنید.
- فایل ZIP دانلود شده را از حالت فشرده خارج کنید.
- داخل آن، پوشه‌هایی بر اساس دسته‌بندی (video، audio، image و ...) وجود دارد.  
  درون هر پوشه، فایل‌های `archive.7z.001`، `archive.7z.002` و ... را می‌بینید.
- با نرم‌افزارهای استخراج مثل **ZArchiver** (اندروید) یا **WinRAR** / **7-Zip** (ویندوز) فایل `archive.7z.001` را باز کنید.  
  این نرم‌افزارها به‌طور خودکار تمام تکه‌ها را به هم می‌چسبانند و فایل اصلی را با **نام و پسوند صحیح** بیرون می‌دهند (بدون نیاز به تغییر دستی).
